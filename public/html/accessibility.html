Accessibility Statement

SLS-U Matter is committed to making this website as accessible and usable as possible for all students, including those with disabilities.

Our commitment

We aim to provide an inclusive digital experience for all users. This includes students with visual, hearing, motor, cognitive, or other disabilities who may use assistive technologies such as screen readers, keyboard navigation, or screen magnifiers.

Accessibility features

Examples of accessibility considerations on SLS-U Matter include:

  • Clean, consistent layout and typography
  • Semantic HTML structure to improve screen reader navigation
  • High-contrast text and backgrounds where possible
  • Responsive design for use on different screen sizes
  • Descriptive link text and button labels where feasible

We are working towards alignment with widely recognized guidelines such as the Web Content Accessibility Guidelines (WCAG) as far as is reasonable for this project.

Keyboard and screen reader use

We aim to ensure that key parts of the site can be navigated using only the keyboard and that important interactive elements provide appropriate labels or roles for assistive technology.

Known limitations

While we strive for accessibility, some areas may still present challenges, such as:

  • Third-party content or embedded media that we do not fully control
  • Complex visual components that may not yet have full ARIA support
  • Occasional missing alternative text or labels

We are committed to gradually improving these areas based on available resources and user feedback.

Feedback and support

If you experience barriers while using SLS-U Matter, or if you have suggestions to improve accessibility, we encourage you to let us know. Please include:

  • The page or feature where you encountered the issue
  • Which device, browser, and assistive technology (if any) you were using
  • A brief description of the problem

You can share feedback via the Contact or Feedback pages, or by contacting the SLSU Guidance and Counseling Office. Together, we can make this platform more inclusive for everyone.

Emergency and alternative access

If accessibility issues prevent you from using this site to request help and you are in immediate distress or danger, please use an alternative method such as:

  • Calling local emergency services
  • Visiting the Guidance and Counseling Office in person during office hours
  • Reaching out to a trusted friend, family member, or staff member
